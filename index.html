Summary: Latitude vs. X

For the Python API Challenge, the task was to generate a list of 500+ unique cities using a random list of latitude and longitude coordinates and the Python Library Citipy.

From this list of cities and the Openweather API, the next task was to find the Max Temperature (F), Humidity (%), Cloudiness (%) and Wind Speed (mph) of these cities and compare them to the latitude.
